# AutoBlog 상품 정보 추출기 (크롬 확장)

쿠팡처럼 크롤링이 차단된 쇼핑몰도, 실제 브라우저로 페이지를 보고 있을 때 이 확장으로 정보를 추출할 수 있습니다.

## 설치 방법

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 "개발자 모드" 켜기
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 `chrome-extension` 폴더 선택

## 사용 방법

1. 쿠팡(또는 다른 쇼핑몰) 상품 페이지로 이동
2. 브라우저 툴바의 확장 아이콘 클릭
3. "클립보드에 복사되었습니다" 알림이 뜨면 성공
4. AutoBlog GUI의 "제휴 글 생성" 탭에서 **"클립보드에서 붙여넣기"** 버튼 클릭 → 상품명/가격/설명/이미지가 자동으로 채워짐
